# NeuroPulse-
 
 ------------------------------------------------------------------------------------------
NEURO-PULSE  is a cutting-edge, futuristic web-based audio visualizer designed for high-performance sound analysis and creative media capture. Developed with a focus on Cybernetic Aesthetics and Neural synchronization, it allows users to transform ambient sound or music files into a holographic light show.

 Features
 ------------------------------------------------------------------------------------------
Dual Input Modes: Seamlessly switch between real-time Microphone capture and MP3/Audio file uploads.

Neural Render Engines:

Radial Mode: A circular holographic burst ideal for bass-heavy tracks.

Linear Mode: A high-precision oscilloscope spectrum for melodic analysis.

Integrated Media Recorder: Capture your visuals and audio simultaneously. Exports high-quality .webm video files directly to your device.

Real-time Calibration: Dynamic sensitivity slider to adjust responsiveness on the fly.

Mobile Optimized: Fully responsive UI that adapts for touchscreens and mobile browsers (iOS/Android).

Security Protocol: A dedicated Terminate button to instantly kill all audio streams and clear memory.

 Tech Stack
 ------------------------------------------------------------------------------------------
HTML5: Semantic structure and Canvas API.

CSS3: Advanced Glassmorphism, Neon UI kits, and CSS Grid/Flexbox for responsiveness.

JavaScript (ES6+):

Web Audio API: For real-time frequency analysis.

MediaRecorder API: For merging canvas video and audio streams into a single file.

Canvas API: For high-frame-rate visual rendering.
